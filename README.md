# My Linux dot files

## Install
Upon first clone of this repo run `setup.sh` to create symlinks in your home directory that point to the dotfiles in this repo.

## Install Vundle
`git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`
